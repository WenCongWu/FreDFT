# FreDFT: Frequency Domain Fusion Transformer for Visible-Infrared Object Detection
This is an official PyTorch implementation for DRANet image denoising.

### 1. Dependences
 Create a conda virtual environment and activate it, e.g.,
 1) conda create --name MOD python=3.9
 2) conda actiavte MOD
 3) pip install -r requirements.txt
